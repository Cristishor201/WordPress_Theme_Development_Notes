### User snippets

ctrl + shift + p -> user snippets -> CSS

```
"media query" : {
	"prefix" : "mq",   // cum apelezi snippet-ul
	"body" : [
		"@media (min-width: $1) {\n\t$0\n}"
	]
}
```

\n - new line
\t -tab
